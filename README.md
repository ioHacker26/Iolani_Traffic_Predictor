### Background
Our project hopes to predict the traffic for the Upper School Autoline during morning hours from approximately 6:50-7:50am. The data has been pulled from security cameras from several points of the drive way from exit to entrance. This data was then tabulated and used in order to train a model which predicts how long for a given car go through.
